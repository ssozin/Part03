# 실습: 브랜드 문서 → 웹사이트 PRD 변환 가이드

브랜드 전략 문서(PDF)를 분석해서 웹사이트용 PRD로 변환하는 방법을 배웁니다.
아래 단계를 따라가며 채운 후, Antigravity에 붙여넣으세요.

---

## STEP 1. 브랜드 문서에서 핵심 추출하기

브랜드 PDF를 열고, 아래 6가지를 찾아 정리하세요.

```
① 브랜드 이름:
② 슬로건/태그라인:
③ 브랜드 소개 (1~2문장):
④ 핵심 키워드: (예: 고급, 자연, 전통, 현대적)
⑤ 타겟 고객: (예: 20~40대 여성, 향수 애호가)
⑥ 제품/서비스: (예: 프리미엄 향수 3종)
```

## STEP 2. 디자인 시스템 추출하기

PDF에서 시각적 요소를 추출합니다.

```
[컬러]
- 메인 컬러: (PDF에서 가장 많이 쓰인 색 → 스포이트로 추출)
- 보조 컬러: (강조나 배경에 쓰인 색)
- 배경 컬러: (밝은 계열? 어두운 계열?)

[폰트]
- 제목용: (PDF에서 쓰이는 큰 글씨 스타일 → 세리프? 산세리프?)
- 본문용: (깔끔한 산세리프 추천)
- 영문 폰트: (브랜드에 사용된 영문 폰트)
- 한글 폰트: (없으면 Pretendard 추천)

[톤 & 무드]
- 전체 분위기: (예: 고급스럽고 차분한 / 활기차고 현대적인)
- 이미지 스타일: (예: 미니멀 제품샷 / 감성적 라이프스타일)
```

## STEP 3. 웹사이트 구조 잡기

브랜드 문서 내용에 맞게 섹션을 구성합니다.

```
[필수 섹션]
- Hero: 브랜드 로고 + 슬로건 + 대표 이미지
- About: 브랜드 스토리 / 철학 (1~2단 레이아웃)
- Products: 제품 라인업 (카드형 그리드)
- Contact: 연락처 + CTA 버튼

[선택 섹션] (브랜드에 맞게 추가)
- Values: 브랜드 핵심 가치 (아이콘 + 설명)
- Gallery: 제품/브랜드 이미지 갤러리
- Testimonial: 고객 후기
- Press: 미디어 소개
```

---

## STEP 4. Antigravity에 보낼 PRD 프롬프트

아래 템플릿을 채워서 Antigravity에 붙여넣으세요.

```
이 브랜드 정보를 기반으로 웹사이트를 만들어줘.

[브랜드 정보]
- 이름: (STEP 1에서 추출)
- 슬로건: (STEP 1에서 추출)
- 소개: (STEP 1에서 추출)
- 핵심 키워드: (STEP 1에서 추출)

[디자인 시스템]
- 메인 컬러: #_____ , 보조 컬러: #_____
- 배경: (밝은/어두운)
- 제목 폰트: _____ , 본문 폰트: _____
- 톤: (고급/캐주얼/전문적/감성적)

[웹사이트 구조]
- Hero: 로고 + 슬로건 + 대표 이미지 (전체 화면)
- About: 브랜드 스토리 (2단 레이아웃)
- Products: 제품 카드 3개 (이미지 + 이름 + 설명 + 가격)
- Contact: 주소 + 이메일 + CTA 버튼

[기술]
- HTML + CSS + 약간의 JS
- 반응형 (모바일 대응)
- 스크롤 애니메이션 적용
```

## 예시: Deeponde 브랜드 PRD

실제 브랜드 문서(Deeponde Fragrance Strategy Deck)를 변환한 예시입니다.

```
이 브랜드 정보를 기반으로 향수 브랜드 웹사이트를 만들어줘.

[브랜드 정보]
- 이름: DEEPONDE (디폰드)
- 슬로건: "깊은 물결, 새로운 시작"
- 소개: 한국의 전통 미학에서 영감을 받은 프리미엄 프래그런스 브랜드. 
  달항아리, 한지, 자연에서 모티프를 가져와 현대적으로 재해석합니다.
- 핵심 키워드: 전통, 자연, 깊이, 고요, 프리미엄

[디자인 시스템]
- 메인 컬러: #1B3A5C (딥 네이비)
- 보조 컬러: #E8D5B5 (웜 베이지), #8B7355 (골드 브라운)
- 배경: 밝은 크림 (#FAF7F2)
- 제목 폰트: Playfair Display (세리프, 고급스러운 느낌)
- 본문 폰트: Pretendard (깔끔한 한글)
- 톤: 고급스럽고 차분하며, 전통과 현대가 공존하는

[웹사이트 구조]
- Hero: 제품 이미지 + 브랜드명 + 슬로건 (전체 화면, 페이드인 애니메이션)
- Brand Story: 브랜드 철학과 영감의 원천 (이미지 + 텍스트 2단)
- Signature Line: 향수 3종 소개 (카드형, 호버 시 설명 표시)
  · The Moon Jar — 달항아리에서 영감, 차분한 플로럴
  · The Hanji — 한지의 질감, 우디 앰버
  · The Garden — 한국 정원, 프레시 그린
- Values: 브랜드 가치 3가지 (아이콘 + 짧은 설명)
- Contact: 구매처 안내 + 뉴스레터 구독 CTA

[기술]
- HTML + CSS + JS
- 반응형 (모바일 대응)
- 스크롤 시 섹션별 페이드인 애니메이션
- 제품 카드 호버 효과
```

## 수정 프롬프트 예시

| 원하는 변경 | 프롬프트 |
|------------|---------|
| 색상 톤 변경 | "전체적으로 더 따뜻한 톤으로 바꿔줘, 베이지 중심으로" |
| 폰트 변경 | "제목 폰트를 Cormorant Garamond로 바꿔줘" |
| 구조 변경 | "Brand Story를 Hero 바로 아래로 옮겨줘" |
| 제품 카드 | "제품 카드를 가로 슬라이드로 바꿔줘" |
| 이미지 추가 | "Hero에 자동 슬라이드 배경 이미지 3장 넣어줘" |
| 애니메이션 | "스크롤할 때 왼쪽에서 슬라이드인 되게 해줘" |
| CTA 강조 | "뉴스레터 구독 버튼을 더 눈에 띄게 만들어줘" |

---

## 팁

- **PDF가 영문이어도 OK** — Antigravity에게 "한국어로 번역해서 적용해줘"라고 하면 됩니다
- **색상을 모르겠으면** — "PDF에서 사용된 컬러를 참고해서 비슷한 톤으로 해줘"
- **이미지가 없으면** — "브랜드 분위기에 맞는 placeholder 이미지를 넣어줘" 또는 이미지 생성 MCP 활용
- **한 번에 완벽할 필요 없음** — 먼저 구조를 잡고, 색상 → 폰트 → 이미지 → 애니메이션 순서로 다듬으세요
- **브랜드 PDF를 Antigravity에 직접 첨부**할 수도 있습니다 — "이 PDF를 참고해서 웹사이트 만들어줘"
