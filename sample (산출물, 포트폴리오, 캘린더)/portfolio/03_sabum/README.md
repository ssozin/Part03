## ✨ Features

| 기능 | 설명 |
|------|------|
| 🖥️ DOS 터미널 UI | 레트로 CRT 모니터 느낌의 인터페이스 |
| 🎮 픽셀 캔버스 애니메이션 | 걸어다니는 캐릭터 + 오브젝트 인터랙션 |
| 🔢 픽셀 숫자 카운터 | 슬롯머신 스타일 글리치 카운트업 |
| 📺 CRT 글리치 이펙트 | 랜덤 스캔라인 왜곡 루핑 |
| 🎨 3가지 테마 | Blue / Black / Prism 배경 전환 |
| 🌐 한영 전환 | KO ↔ EN 원클릭 언어 스위칭 |
| 🖼️ 픽셀아트 갤러리 | 포트폴리오 이미지 도트 변환 |
| 🖱️ 레트로 커서 | 픽셀 마우스 포인터 |

## 🛠️ Tech Stack

- **HTML / CSS / JavaScript** (No Framework)
- Canvas API (Pixel Animation)
- CSS Custom Properties (Theme System)
- IntersectionObserver (Scroll Reveal)
- LocalStorage (Theme & Language Persistence)
